基本使用 #

BackTop 的基本使用

返回顶部
html
<f-back-top>返回顶部</f-back-top>
展开代码

不同滚动模式 #

behavior 属性可以配置不同的滚动模式

auto 模式
smooth 模式
html
<f-back-top bottom="90px" behavior="auto">auto 模式</f-back-top>
<f-back-top bottom="140px" behavior="smooth">smooth 模式</f-back-top>
展开代码

圆形的 #

round 属性可以配置圆形的样式

html
<f-back-top bottom="190px" round>
  <f-svg-icon :icon="FIconSnowflake" />
</f-back-top>

<script lang="ts" setup>
  import { FIconSnowflake, FIconUpload } from '@fighting-design/fighting-icon'
</script>

自定义样式 #

你也可以自定义 back-top 的样式和位置

html
<f-back-top bottom="190px" right="100px" round color="red">
  <f-svg-icon :icon="FIconUpload" />
</f-back-top>

Attractive #

参数说明类型可选值默认值
round是否为圆角boolean——false
behavior滚动模式stringsmooth autosmooth
visible-height滚动超出多少距离展示number——200
right距离右侧的距离string——40
bottom距离底部的距离string——40
z-index原生 z-indexnumber——200
top点击返回距离顶部的距离number——0
listen-el监视指定元素，需要传入指定的 class 或 id，如：.box #appstring————
background背景色string————
color文字颜色string————
