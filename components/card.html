基本使用 #

title 属性可以配置卡片的标题

卡片标题
Card 卡片
html
<f-card title="卡片标题">Card 卡片</f-card>

圆角卡片 #

round 是否为圆角卡片

圆角的
Card 卡片
html
<f-card round title="圆角的">Card 卡片</f-card>
展开代码

不同状态 #

shadow 属性可以配置阴影样式

从不显示
Hover 显示
总是显示
html
<f-card shadow="never">从不显示</f-card>
<f-card shadow="hover">Hover 显示</f-card>
<f-card shadow="always">总是显示</f-card>
展开代码
